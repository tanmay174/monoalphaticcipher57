Encryption uses an algorithm and a key to transform an input
(i.e., plaintext) into an encrypted output (i.e., ciphertext). Algorithms are considered secure if an attacker cannot determine any properties of the plaintext or key.
An attacker should not be able to determine anything about a key given a large number of plaintext/ciphertext combinations which used the key.
